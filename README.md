
# CodeClause Data Science Internship Projects

This repository contains projects completed during the CodeClause Data Science Internship.

## Project 1: Movie Review Sentiment Analysis
- Uses NLP and Machine Learning to analyze movie reviews
- Dataset: IMDB Movie Reviews Dataset
- Dataset Source: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

## Project 2: Movie Genre Prediction
- Predicts movie genre from plot summaries
- Dataset Source: https://www.kaggle.com/datasets/jrobischon/wikipedia-movie-plots

## Project 3: Customer Segmentation
- Customer segmentation using K-Means clustering
- Dataset Source: https://www.kaggle.com/datasets/shwetabh123/mall-customers

## Golden Project: Heart Disease Risk Assessment
- Predicts heart disease risk using machine learning
- Dataset Source: UCI Heart Disease Dataset
